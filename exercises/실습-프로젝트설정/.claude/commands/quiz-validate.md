---
description: questions.js에서 최상급 표현이 포함된 퀴즈 문제를 찾아 목록으로 출력
---

`questions.js` 파일을 읽어, `question` 필드에 다음 최상급 표현 중 하나 이상이 포함된 문제를 모두 찾아 목록으로 보여줘.

**검색 키워드**
- 가장
- 최초
- 최대
- 최고
- 최소
- 최저
- 제일
- 첫 번째
- 마지막

**출력 형식**

발견된 각 문제를 아래 형식으로 나열:

```
[id] (카테고리) 매칭 키워드: <키워드>
   질문: <question 본문>
```

마지막에 총 발견 개수를 카테고리별 집계와 함께 표시.

**참고**
- 단순 키워드 검색이며, 문맥상 최상급이 아닌 경우(예: '가장자리')도 매칭될 수 있으므로 결과 확인 후 사용자가 판단하도록 안내.
- CLAUDE.md의 "퀴즈 문제 교차 검증 가이드라인" 2번 항목("최상급 표현에 기준이 있는가?")과 대조하여, 측정 기준이 명시되지 않은 문제는 별도로 표시.
