description: questions.js의 난이도 분포와 정답 위치 분포를 분석하여 출제 균형을 점검
---

`questions.js` 파일을 읽고, 퀴즈의 **난이도 분포**와 **정답 위치 분포**를 분석해줘.

## 분석 항목

### 1. 난이도 분포 (`difficulty` 필드)
- 전체 문제 수
- `easy` / `medium` / `hard` 각 개수와 비율(%)
- 카테고리별 난이도 분포 (카테고리 × 난이도 매트릭스)

### 2. 정답 위치 분포 (`correctAnswer` 필드)
- 0번(첫 번째) / 1번 / 2번 / 3번(네 번째) 선택지가 정답인 문제 수와 비율(%)
- 카테고리별 정답 위치 분포
- 난이도별 정답 위치 분포

### 3. 균형 점검
- 난이도 비율이 한쪽으로 치우쳤는지 (권장: easy 30~40% / medium 30~40% / hard 25~35% 부근)
- 정답 위치가 특정 인덱스에 편중되어 있는지 (이상적으로는 각 25% 부근)
- 같은 정답 인덱스가 3문제 이상 연속으로 등장하는 구간이 있는지 (있다면 해당 id 구간 표시)
- 카테고리별 문제 수 격차가 큰지

## 출력 형식

```
■ 전체 문제 수: N

■ 난이도 분포
   easy   : XX개 (XX.X%)
   medium : XX개 (XX.X%)
   hard   : XX개 (XX.X%)

■ 카테고리별 난이도
   카테고리        easy  medium  hard  합계
   한국사            X      X     X     X
   세계지리          X      X     X     X
   ...

■ 정답 위치 분포 (correctAnswer)
   index 0 : XX개 (XX.X%)
   index 1 : XX개 (XX.X%)
   index 2 : XX개 (XX.X%)
   index 3 : XX개 (XX.X%)

■ 카테고리별 정답 위치
   카테고리        0    1    2    3
   한국사          X    X    X    X
   ...

■ 난이도별 정답 위치
   easy   : 0=X, 1=X, 2=X, 3=X
   medium : 0=X, 1=X, 2=X, 3=X
   hard   : 0=X, 1=X, 2=X, 3=X

■ 균형 점검 결과
   - 난이도 균형: <양호 / 치우침 — 설명>
   - 정답 위치 균형: <양호 / 편중 — 설명>
   - 연속 동일 정답: <없음 / 발견된 구간 (id A → id B, 정답 index = X)>
   - 카테고리 문제 수 격차: <양호 / 큼 — 설명>

■ 개선 제안
   - <필요한 경우에만 구체적인 보강 방향>
```

## 참고

- 비율은 소수점 첫째 자리까지 표시.
- "연속 동일 정답"은 `questions.js` 파일에 등장하는 순서 기준 (id 순이 아니라 파일 내 배열 순서).
- 편중 판정은 단순 비율 외에 카이제곱 같은 통계 검정까지는 요구하지 않음. 직관적 판단으로 충분.
- 분포가 모두 양호하면 "균형 양호" 한 줄로 짧게 마무리해도 됨.
